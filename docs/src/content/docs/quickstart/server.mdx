<Steps>

1. Create a hono.js project using nodejs template

   ```bash
   pnpm create hono@latest my-mcp
   ```

2. Install `muppet` and `@modelcontextprotocol/sdk`

    ```bash
    pnpm add muppet @modelcontextprotocol/sdk
    ```

</Steps>
